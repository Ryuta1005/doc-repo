# ドキュメントの編集

doc-repoでは、Markdownドキュメントをブラウザから編集できます。

保存した内容は、元の`.md`ファイルへ直接反映されます。

## Viewerを開く

Markdownドキュメントを管理しているプロジェクトで、次のコマンドを実行します。

```bash
npm run doc-repo
```

起動後、ターミナルに表示されたURLをブラウザで開きます。

デフォルトでは、次のURLからアクセスできます。

```text
http://localhost:4000
```

詳しい起動方法や設定については、[Getting Started](./getting-started.ja.md)を参照してください。

## 編集

1. 左側のドキュメントツリーから編集するファイルを選択します。
2. 画面右上の **Edit**をクリックします。
3. ドキュメントを編集します。
4. 必要に応じて本文の上にあるファイル名欄も編集します。
5. 編集が終わったら **Save**をクリックします。

編集内容を保存せずに終了する場合は、**Cancel**をクリックします。

## 新規作成

1. ドキュメントツリーでファイルまたはフォルダ行をホバーします。
2. ホバー行に表示された `+` ボタンをクリックします。
3. 編集画面へ直接遷移し、ツールバーと本文入力欄の間にファイル名入力欄が表示されます。
4. ファイル名を入力して **保存** をクリックします。この時点でファイルが新規作成されます。

特記事項:

- 入力名には常に `.md` が付与されます。例: `doc.ja` → `doc.ja.md`、`a.md` → `a.md.md`
- パス区切り文字やパスとして解釈される入力は拒否されます。

## 文書・フォルダの削除

1. ドキュメントツリーでファイルまたはフォルダ行をホバーします。
2. 行末のミートボールメニュー（Ellipsis）をクリックします。
3. ゴミ箱アイコン付きの **削除** を選択します。
4. 確認ダイアログで対象名と警告文を確認します。
5. **削除** で実行、**キャンセル** で中止します。

特記事項:

- 削除はファイル・フォルダの両方で利用できます。
- フォルダ削除は Policy B に従い、配下に管理対象外またはスコープ外項目が 1 つでもある場合は拒否されます。
- 未保存変更がある場合は、削除確認より先に既存の未保存変更確認が表示されます。
- 削除成功後はツリーが再読み込みされ、残存文書または空状態へ遷移します。

## 保存

**Save**をクリックすると、編集内容が現在開いている`.md`ファイルへ保存されます。

編集画面でファイル名を変更した場合は、同じフォルダ内の新しい`.md`ファイル名へ保存され、古い名前は削除されます。

doc-repoは保存前に、保存対象が次の条件を満たしていることを確認します。

- `rootDir`配下にある
- 既存のMarkdownファイルである
- `include`と`exclude`の条件を満たしている

条件を満たしていない場合、ファイルは変更されず、画面にエラーが表示されます。

## 対応書式

リッチテキストエディタは、次の書式に対応しています。

- 本文
- 見出し1〜6
- 太字
- イタリック
- 取り消し線
- インラインコード
- 引用
- 箇条書き
- 番号付きリスト
- コードブロック
- 分割線
- 画像のアップロードと挿入
- Viewerで使用する`info panel` / `admonition`

Viewerでは、リッチテキストエディタが対応していないMarkdownも表示できる場合があります。対応していないMarkdownを含むドキュメントは、編集・保存によって内容が変化する可能性があります。

## キーボードショートカット

| 操作             | macOS    | Windows / Linux |
| ---------------- | -------- | --------------- |
| 本文             | `⌘⌥0`    | `Ctrl+Alt+0`    |
| 見出し1          | `⌘⌥1`    | `Ctrl+Alt+1`    |
| 見出し2          | `⌘⌥2`    | `Ctrl+Alt+2`    |
| 見出し3          | `⌘⌥3`    | `Ctrl+Alt+3`    |
| 見出し4          | `⌘⌥4`    | `Ctrl+Alt+4`    |
| 見出し5          | `⌘⌥5`    | `Ctrl+Alt+5`    |
| 見出し6          | `⌘⌥6`    | `Ctrl+Alt+6`    |
| 太字             | `⌘B`     | `Ctrl+B`        |
| イタリック       | `⌘I`     | `Ctrl+I`        |
| 取り消し線       | `⌘⇧S`    | `Ctrl+Shift+S`  |
| インラインコード | `⌘⇧M`    | `Ctrl+Shift+M`  |
| 引用             | `⌘⇧B`    | `Ctrl+Shift+B`  |
| 箇条書き         | `⌘⇧8`    | `Ctrl+Shift+8`  |
| 番号付きリスト   | `⌘⇧7`    | `Ctrl+Shift+7`  |
| コードブロック   | `⌘⌥C`    | `Ctrl+Alt+C`    |
| 分割線           | `⌘⇧-`    | `Ctrl+Shift+-`  |
| 保存             | `⌘Enter` | `Ctrl+Enter`    |
