# 021 文書を作成削除できる

## メタ情報

- 種別: Epic
- 番号: 021
- ステータス: Todo
- 優先度: P2
- Spec: -
- Depends On: 010

## 価値

利用者がブラウザ上のワークスペースだけで Markdown 文書の追加と整理を完結できるようにする。既存文書の編集だけでなく、新しいページを作り、不要になったページを削除できることで、日常的なドキュメント管理に近づける。

## 受け入れ条件の要点

- Story 022, 023 が Done である
- 新しい Markdown 文書を rootDir 配下に作成できる
- 不要な Markdown 文書を確認後に削除できる
- 作成・削除後に文書ツリーと表示状態が更新される
- rootDir 外、`.md` 以外、include / exclude 対象外への操作を拒否できる
- 未保存変更がある状態で作成・削除操作を行う場合、既存の未保存変更ガードを破壊しない
