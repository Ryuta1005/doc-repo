# 016 編集保存を安全に運用できるようにする

## メタ情報

- 種別: Epic
- 番号: 016
- ステータス: Todo
- 優先度: P2
- Spec: -
- Depends On: 009

## 価値

ブラウザ上の編集・保存を日常利用できる品質に引き上げる。外部変更との競合や保存失敗を利用者が理解し、誤上書きや作業内容の喪失を避けられるようにする。

## 受け入れ条件の要点

- Story 011, 013 が Done である
- 外部変更を検知し、未保存変更を自動で破壊しない
- 保存時の不一致を競合として扱い、誤上書きを防止できる
- 保存できなかった理由と推奨アクションを利用者が確認できる
- 失敗後も編集内容を保持し、再試行や復旧を選択できる
