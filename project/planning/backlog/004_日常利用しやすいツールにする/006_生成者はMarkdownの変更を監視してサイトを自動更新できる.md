# 生成者はMarkdownの変更を監視してサイトを自動更新できる。なぜなら毎回コマンドを再実行せずに最新の内容をブラウザで確認したいから。

- Epic: 004
- ステータス: Done

- Spec: 006-markdown-watch
- Depends On: 005

## ユーザー価値

`doc-repo serve` 実行中に Markdown を編集・保存すると、手動操作なしでブラウザに最新内容が反映される。

## 受け入れ条件

- `doc-repo serve` 実行中に `.md` ファイルを変更・保存すると自動で再生成が行われる
- 再生成完了後、ブラウザが自動更新される（ホットリロード）
- 新規 `.md` ファイルの追加も変更検知の対象となる
- `.md` ファイルの削除も変更検知の対象となる
- 変更検知から再生成・ブラウザ更新までの状態が標準出力から把握できる

## 実装タスク管理

- 機能実装タスクは Speckit のフローで管理する
