## ユーザー価値

利用者は閲覧中のMarkdownをブラウザ上で編集し、元のMarkdownファイルへ保存できる。なぜなら、外部エディタへ移動せず、その場でドキュメントを修正したいから。

## 受け入れ条件

- 閲覧画面から編集モードへ切り替えられる
- 選択中の Markdown 本文が編集画面へ表示される
- Markdown 本文をブラウザ上で編集できる
- 編集中の内容をブラウザ内で保持できる
- 編集内容を元の Markdown ファイルへ保存できる
- 保存成功後、最新内容がプレビューへ反映される
- 編集を破棄して閲覧画面へ戻れる
- 保存対象は解決済み rootDir 配下の Markdown に限定される
- `.md` 以外のファイルへの保存を拒否する
- include 対象かつ exclude 対象外のファイルのみ保存できる
- `..` 等によるパストラバーサルを拒否する
- 保存成功・保存失敗を利用者が確認できる
