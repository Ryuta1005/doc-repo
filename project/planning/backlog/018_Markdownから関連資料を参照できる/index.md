# 018 Markdownから関連資料を参照できる

## メタ情報

- 種別: Epic
- 番号: 018
- ステータス: Todo
- 優先度: P3
- Spec: -
- Depends On: 014

## 価値

Markdown 本文だけでなく、PDF や CSV などの関連資料もドキュメントサイト上の自然なリンクとして参照できるようにする。閲覧・共有体験を Markdown 以外の資料まで広げる。

## 受け入れ条件の要点

- Story 015 が Done である
- Markdown の通常リンクからリポジトリ内の非Markdownファイルを開ける
- 添付ファイルは生成サイトから参照できる場所へコピーされる
- rootDir 外やパストラバーサルはコピー対象にしない
- Markdown ページリンクや外部URLなど既存リンクの扱いを壊さない
