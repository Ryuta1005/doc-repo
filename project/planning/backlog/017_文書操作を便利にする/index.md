# 017 文書操作を便利にする

## メタ情報

- 種別: Epic
- 番号: 017
- ステータス: Todo
- 優先度: P3
- Spec: -
- Depends On: 009

## 価値

編集・保存の基盤の上に、文書内の軽量な状態更新や構造化編集を積み上げる。チェックボックスの状態保持や表編集をできるようにし、進捗管理・情報整理などの日常的な文書操作をしやすくする。

## 受け入れ条件の要点

- Story 012 が Done である
- Story 024 が Done である
- Story 025 が Done である
- 文書内チェックボックスの変更を保持できる
- 再表示後も保持された状態を確認できる
- 保持処理で他の本文内容を破壊しない
- 表を挿入・編集して Markdown として保存できる
- 保存後の再表示でも表の内容が保持される
- サイドバーの作成導線からファイルまたはフォルダを選択して作成できる
- 作成された空フォルダがサイドバー再読込後も表示される
